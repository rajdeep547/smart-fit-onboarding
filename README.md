## ✨ Features

### 📝 Manual Flow
- ✅ Clean, mobile-first form experience
- ✅ All 10 Fit Quiz questions
- ✅ Conditional brand sizing (Q9 follows Q8)
- ✅ Optional weight with easy skip
- ✅ Progress tracking
- ✅ Smooth navigation with back/edit

### 🎙️ AI Voice Flow
- ✅ Voice-to-voice conversation using Web Speech API
- ✅ Natural language understanding
- ✅ Spoken questions with confirmation
- ✅ Graceful error handling
- ✅ Skip optional questions naturally
- ✅ Works on Chrome & Safari (mobile/desktop)

### 🔄 Handoff
- ✅ Fit profile saved to localStorage
- ✅ Data passed via URL parameters
- ✅ Redirect to main site: [jackie-jeans.vercel.app](https://jackie-jeans.vercel.app/)

## 🚀 Quick Start

### Prerequisites
- Node.js 16+ ([Download](https://nodejs.org/))
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/rajdeep547/smart-fit-onboarding.git

# Navigate to project
cd smart-fit-onboarding

# Install dependencies
npm install

# Run development server
npm run dev
